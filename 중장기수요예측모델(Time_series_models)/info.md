# <중장기 수요예측모델>  
  
시계열모델 비교  
  
- SARIMAX
- GRU
- BILSTM  
  
중장기 수요예측모델을 위한 분석용 데이터셋은  
데이터 마트에 있는 각 수주건에 대한 날짜, 판매량, 제품명 데이터를 제품별 일별 판매량 합계로 변경하여 사용  
** GRU model은 날씨 데이터의 기온데이터와 습도데이터를 함께 고려하여 분석  

RMSE와 $R**2$ 기준으로 성능이 좋았던 케라스의 GRU 모델을 최종모델로 선정.
